Инструкция по использованию приложения
Запуск приложения:

Запустите приложение. Откроется графический интерфейс с несколькими полями и кнопками.
Выбор файлов:

Нажмите на кнопку "Выбрать первую таблицу".
В появившемся диалоговом окне выберите файл Excel, содержащий исходные данные.
Нажмите на кнопку "Выбрать вторую таблицу".
В появившемся диалоговом окне выберите файл Excel, в который будут вноситься изменения.
Выбор книги (листа):

В выпадающем списке "Выберите книгу из первого файла:" выберите нужную книгу из первой таблицы.
В выпадающем списке "Выберите книгу из второго файла:" выберите нужную книгу из второй таблицы.
Выбор распределительного центра и даты:

В выпадающем списке "Выберите распределительный центр:" выберите нужный распределительный центр.
В выпадающем списке "Выберите дату:" выберите нужную дату.
Обработка данных:

Нажмите кнопку "Обработать данные" два раза.
Приложение сравнит данные из выбранной первой таблицы с данными из второй таблицы по выбранному распределительному центру и дате.
Результаты обработки:

Статус обработки:

В нижней части окна приложения будет отображаться статус обработки запроса:
"Запрос обработан" (зеленый цвет) — запрос успешно обработан.
"Ошибка: Количество паллет не совпадает!" (красный цвет) — количество паллет в двух таблицах не совпадает.
"Ошибка: Нет данных для выбранного распределительного центра и даты." (красный цвет) — нет данных для указанного распределительного центра и даты.
"Ошибка: В первой таблице отсутствует колонка 'New_Pallets'." (красный цвет) — в первой таблице отсутствует необходимая колонка.
"Ошибка: В целевой таблице отсутствуют необходимые колонки." (красный цвет) — в целевой таблице отсутствуют необходимые колонки.
Выделение ячеек:

Количество паллет в целевой таблице будет выделено:
Зеленым цветом, если количество совпадает с данными из первой таблицы.
Красным цветом, если количество не совпадает.
Проверка и завершение:

Убедитесь, что статус обработки отображает правильное состояние.
Закройте приложение, когда завершите работу.
